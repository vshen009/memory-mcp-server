### 集成方案

**候选方案:**

1. **直接 API 调用**
   - 优点: 简单直接
   - 缺点: 需要处理错误和重试

2. **SDK 集成**
   - 优点: 官方支持
   - 缺点: 依赖第三方

3. **Webhook 模式**
   - 优点: 实时性好
   - 缺点: 需要公网服务器
